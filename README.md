# 📊 Task 3: SQL for Data Analysis

## 📌 Overview

This repository contains my submission for **Task 3: SQL for Data Analysis** as part of the **Elevate Labs Data Analyst Internship**.

The objective of this task was to use SQL queries to extract, manipulate, and analyze structured data from a Walmart sales dataset while demonstrating fundamental SQL concepts used in data analysis.

---

## 🎯 Objective

* Extract meaningful insights from a structured dataset using SQL.
* Practice data retrieval, filtering, aggregation, and analytical queries.
* Demonstrate the use of SQL for business analysis.

---

## 🛠️ Tools Used

* MySQL
* MySQL Workbench

---

## 📂 Dataset

* **Dataset:** Walmart Sales Dataset
* **Format:** CSV

---

## ✅ SQL Concepts Covered

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* HAVING
* Aggregate Functions (`SUM`, `AVG`, `COUNT`, `MIN`, `MAX`)
* CASE Statements
* Subqueries
* Views
* Indexes
* Business Analysis Queries

---

## 📊 Business Analysis Performed

* Calculated total transactions.
* Analyzed sales quantity by branch and city.
* Compared category performance.
* Evaluated payment method usage.
* Calculated revenue using `Unit Price × Quantity`.
* Identified top-performing branches and cities.
* Measured customer ratings across categories.
* Generated daily sales summaries.
* Created reusable SQL views for reporting.
